## Install

```sh
sudo apt install git python3
git clone https://github.com/Sl-Sanda-Ru/Py-Fuscate.git
cd Py-Fuscate
python3 -m pip install -r requirements.txt
python3 py_fuscate.py
```

## Usage

```sh
usage: py_fuscate.py [-h] -i INPUT -o OUTPUT -c COMPLEXITY
```
